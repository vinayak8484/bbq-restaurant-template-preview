──────────────────────────────────────────────────────────────
  HOW TO GET STARTED
──────────────────────────────────────────────────────────────

  1. Open the HTML file in your browser.
     It works immediately with no installation needed.

  2. To edit the content, open the file in any text editor
     (Notepad, TextEdit, VS Code, Notepad++, etc.)

  3. Use CTRL+F (or CMD+F on Mac) to search for the text
     you want to change.

  4. Save the file and reload the browser to see changes.


──────────────────────────────────────────────────────────────
  THE 5 THINGS TO CHANGE FIRST
──────────────────────────────────────────────────────────────

  1. RESTAURANT NAME
     Search for the example name (Smoke & Bone,
     Lone Star Pit, Ember) and replace it with yours.
     It appears in the navbar logo, hero, footer, and
     browser tab title (<title> tag at the top).

  2. ADDRESS AND CONTACT DETAILS
     Find the example address and type in yours.
     Do the same with phone number and email.
     The address appears in the info strip, footer,
     and map section.

  3. MENU AND PRICES
     Text is hardcoded directly in the HTML. Use CTRL+F
     to find each dish name and update the name, description,
     and price. Prices are in the format $XX or $XX / lb.

  4. OPENING HOURS
     Find the example hours in the info strip section
     and update them. Search for "Wed" or "10:30 AM"
     to locate them quickly.

  5. GOOGLE MAPS
     Go to maps.google.com, search your address, click
     Share and then Embed a map. Copy the iframe code.
     In the HTML, find the <iframe> tag in the map section
     and replace it with yours.


──────────────────────────────────────────────────────────────
  DARK MODE AND LIGHT MODE
──────────────────────────────────────────────────────────────

  Every template has a day / night mode toggle button in the
  navbar. The user's preference is saved automatically.

  Smoke & Bone:   defaults to DARK. Toggle switches to light.
  Lone Star Pit:  defaults to LIGHT. Toggle switches to dark.
  Ember:          defaults to DARK. Toggle switches to light.

  To change a template's default starting mode, find
  the JavaScript section at the bottom of the file and
  look for the localStorage key. Instructions are in
  comments inside the script.


──────────────────────────────────────────────────────────────
  COLOURS
──────────────────────────────────────────────────────────────

  Main colours are defined as CSS variables at the top
  of the file, inside the :root { ... } section.

  To change the main colour, find the corresponding
  variable and enter your colour in HEX format.

  Smoke & Bone example:
    --fire:    #C0392B;   change to your main colour
    --amber:   #E8A21A;   change to your accent colour

  Lone Star Pit example:
    --rust:    #B7410E;   change to your main colour
    --gold:    #D4891A;   change to your accent colour

  Ember example:
    --ember:   #E04A0C;   change to your main colour
    --copper:  #B87333;   change to your accent colour
